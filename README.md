# Hyperinbox
#### 소요기간 : 2021년 06월 01일 ~ 2021년 07월 30일
#### 개발기간 : 2021년 07월 22일 ~ 2021년 07월 28일

## [ 개요 ]

**디자인 협업 소프트웨어 하이퍼인박스 모바일버전**

slack, figma, github등의 협업 툴을 한곳에서 사용할 수 있는 협업 소프트웨어\
기존의 하이퍼인박스 데스크톱 앱을 모바일버전으로 제작 
* 로그인
* 인박스
* 프로젝트
* 검색
* 마이페이지
